# HydroPlay

A free-to-play online gaming website where users can instantly play hundreds of games directly in the browser — no downloads, sign-ups, or installations required.

## Features

- **500+ browser games** powered by [LuminSDK](https://github.com/luminsdk)
- **Instant play** — no downloads or accounts needed
- **Search & filter** to quickly find games
- **Responsive design** — works on desktop, tablet, and mobile
- **Dark theme** with animated gradient background

## Getting Started

No build tools or dependencies required. Just serve the static files:

```bash
# Option 1: Open index.html directly in your browser

# Option 2: Use a local HTTP server
python3 -m http.server 8000
# Then visit http://localhost:8000
```

## Deployment

Deploy `index.html` to any static hosting service (GitHub Pages, Netlify, Vercel, etc.).

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- [LuminSDK](https://github.com/luminsdk) for game integration
- [Inter](https://fonts.google.com/specimen/Inter) font via Google Fonts
